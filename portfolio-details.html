This is a Quiz-App

This project is a full-stack web application built with React for the frontend and FastAPI (Python) for the backend. It provides functionality to generate multiple-choice questions (MCQs) based on a given URL. For now Quiz Generation using URL is implemented. We can extend this to quiz generation from PDFs, i/p text content etc.
